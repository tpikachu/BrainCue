# 0.5.0 — 2026-06-23

First downloadable release — plus region-capture fixes.

## Added

- **Downloadable installers.** BrainCue Copilot is now available as a Windows
  installer (`.exe`), a macOS `.dmg`, and a Linux `.AppImage` from the
  [Releases](https://github.com/tpikachu/Interview-Copilot/releases) page — built
  automatically by CI.

## Fixed

- **"Select region to solve" works reliably.** It could open a blank black screen
  you couldn't draw on; the selector is now pre-loaded so it always shows your
  screenshot and lets you drag a region right away.
- **The region selector is hidden from screen sharing.** While you select an area,
  the capture overlay no longer appears to anyone you're sharing your screen with.
- **App icon.** The window and taskbar now show the BrainCue icon instead of the
  default Electron icon.

## Notes

- Installers are currently **unsigned**, so the OS may warn on first launch
  (Windows SmartScreen → "Run anyway"; macOS Gatekeeper → right-click → Open).
