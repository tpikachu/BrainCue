# 0.2.0 — 2026-06-23

Background-friendly app lifecycle, customizable shortcuts, and a refreshed brand mark.

## Added

- **System tray.** The app now lives in the system tray. Closing the dashboard
  window no longer quits — it keeps running in the background and frees the
  taskbar button. Click the tray icon to bring the dashboard back.
- **Tray menu** with quick actions: Show AI Assistant, Show Overlay, a Privacy
  Mode toggle (checkbox), Settings, and Exit (with a confirmation prompt).
- **Customizable global shortcuts.** A new *Keyboard Shortcuts* section in
  Settings lets you re-bind every global hotkey by clicking and pressing the
  keys you want. Changes apply live (no restart) and can be reset per-shortcut
  or all at once. Global shortcuts are suspended while you're recording a new
  binding so the keystroke reaches the editor instead of firing an action.
- **Animated app logo** in the dashboard header — the brand mark now carries a
  "B" monogram and a professional gradient that sweeps around its edge
  (respects `prefers-reduced-motion`).
- **Single-instance guarantee.** Launching the app a second time focuses the
  existing window (restoring it from the tray) instead of starting a duplicate.

## Changed

- **Quick-solve hotkey** moved from `Ctrl+Shift+Enter` to `Ctrl+Shift+A`. The
  old combo collides with NVIDIA's 3D-viewer hotkey on GeForce/GTX machines.
- **Privacy Mode stays in sync everywhere.** Toggling it from the global
  shortcut, the overlay button, the tray, or Settings now updates all of them at
  once, including the tray checkbox and the Settings switch.

## Fixed

- **No more orphaned processes on exit.** Exiting now performs a single,
  ordered shutdown that closes the live transcription websocket, unregisters
  global shortcuts, destroys the tray, and tears down all windows — so no Electron
  helper processes survive after you quit.

## Security

- Privacy Mode (hide from screen sharing/recording) remains **on by default** and
  is applied to every window, including newly created ones.
