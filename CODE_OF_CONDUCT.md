# Code of Conduct

BrainCue's community — issues, PRs, discussions, and any space tied to this
project — is for building software. Everyone participating is expected to:

- **Be respectful.** No harassment, insults, discrimination, or personal
  attacks. Critique code and ideas, never people.
- **Be honest.** Don't misrepresent what your contribution does, plagiarize
  work, or game evaluation systems. (Contributions are scored — attempting to
  manipulate scoring is treated as bad faith, not cleverness.)
- **Be constructive.** Disagreement is fine; make it specific, technical, and
  actionable. "This breaks X because Y" beats "this is bad."
- **Respect maintainer decisions.** Merge, close, and scope calls are made by
  the maintainers. Appeals are welcome once, with new information.

Unacceptable behavior — including sustained hostility, spam, deliberate
gaming, or retaliation against reporters — results in warnings, and for
repeated or severe cases, bans from participation.

Report conduct issues privately to the maintainer via
[GitHub](https://github.com/tpikachu) or, if it concerns a security matter,
through the process in [SECURITY.md](SECURITY.md). Reports are handled
confidentially.
